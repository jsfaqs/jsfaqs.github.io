DOM Questions

The DOM is a built-in library for JavaScript running in web pages. Not all JavaScript runs in the browser, so not environments have the DOM. As such, questions pertaining to the DOM will not be answered in here.

This also includes questions about jQuery, Angular, Backbone, and other such libraries.
